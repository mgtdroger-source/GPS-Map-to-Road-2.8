Distance Test App v2.8

Used to validate distances between waypoints, either by trip-line detection or by entering a 30 metre spot/radius marker. A full GPS log can be exported for detailed analysis, including GPS performance during the run. Distance errors are reported in metres against the reference distance provided.
Load or create a route before starting. The app will not silently fall back to an old test route.

1. GPS / Test Control

GPS starts when the app opens. Wait for GPS ready, press Start Test before the first line, allow the 30 second settle countdown, then drive through the first line. Measurement starts at the first line crossing.

4. Trip Lines / Splits

Line crossings or 30 m spot/radius hits define measurement points. If line coordinates exist, line detection is used; if only a spot coordinate exists, radius detection is used.

5. Export

Full GPS Log CSV is the main review file and gives every GPS fix and per-fix diagnostic value. Summary CSV is a shorter split/result report and may be empty if no line or radius point was triggered.

6. Route Lines / Manual Entry

Enter each test point. Line Start/End coordinates use line-crossing detection. If only a Spot coordinate is entered, the app uses a 30 m radius hit. Expected/map distance is manual and used only for comparison in exports.
